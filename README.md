## Project Setup
```
 git clone https://github.com/WarrenU/shake-search.git
 cd shake-search
 go run ./main.go
```
(Server should be running on port 9000, next you may curl)

## cURL Request Example
`curl http://localhost:9000/autocomplete?term=th`
